What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for HTML document. It represents HTML document as a tree, where every node is an element, attribute or content. DOM can dynamically modify HTML document's element, content, styling, etc.

How do you select an element from the DOM?

There are 5 way to select an HTML element from the DOM.
1. document.getElementByID()
2. document.getElementsByClassName()
3. document.getElementByTagName()
4. document.querySelector()
5. document.querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is refer to the process of using event bubbling. It is useful because it reduce large number of event listener.

How do you manipulate an element's attributes and styles using the DOM?

Manipulate an element's attributes and styles in the DOM using JavaScript is
1. setAttribute()
2. removeAttribute()
3. style.property = value
4. classList.add()
5. classList.remove()
